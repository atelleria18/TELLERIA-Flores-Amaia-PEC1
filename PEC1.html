  1. Preparacin de la Matriz de Datos y Metadatos

Se extrajo la matriz de datos necesaria para el anlisis, seleccionando las columnas especficas para cada condicin. A continuacin, se cre un marco de datos para los metadatos de las muestras, agrupndolas segn las condiciones experimentales: MSS y PD.

# Prepare data matrix and metadata
data_matrix <- as.matrix(data[ , c("M1_1_MSS", "M1_2_MSS", "M5_1_MSS", "M5_2_MSS", 
                                   "T49_1_MSS", "T49_2_MSS", "M42_1_PD", "M42_2_PD", 
                                   "M43_1_PD", "M43_2_PD", "M64_1_PD", "M64_2_PD")])

# Metadata for rows (samples)
sample_metadata <- data.frame(
  row.names = colnames(data_matrix),
  condition = c(rep("MSS", 6), rep("PD", 6))
)

# Create SummarizedExperiment object
se <- SummarizedExperiment(
  assays = list(counts = data_matrix),
  colData = sample_metadata
)

  1. Transformacin de Datos

Para reducir la variabilidad en los datos, se aplic una transformacin logartmica. Esto permite analizar los datos con una distribucin ms uniforme, facilitando la identificacin de patrones en el heatmap.

# Aplicar una transformacin logartmica a los datos para reducir la variabilidad
log_counts <- log2(assay(se, "counts") + 1)  # +1 para evitar log(0)

# Reemplazar la matriz de conteo en el objeto SummarizedExperiment
assay(se, "counts") <- log_counts
  1. Visualizacin Avanzada

Se gener un heatmap de los datos transformados utilizando la librera pheatmap, con anotaciones para las condiciones.

library(pheatmap)


# Asegrate de que `annotation_col` es un DataFrame de una sola columna

# Generar el heatmap nuevamente

    scale = "row",

Cuando Git solicit las credenciales, ingres mi nombre de usuario de GitHub (atelleria18) y, en lugar de una contrasea, pegu mi token de acceso personal.
